# FogTech Services — Website Platform

Professional disinfecting, sanitizing, and fogging services website platform developed and maintained by STG / SDL.

---

## About This Project

This repository contains the production-ready local development version of the FogTech Services website.

This project was originally generated through an upstream rapid-prototyping workflow and has since been sanitized, hardened, optimized, and migrated into a standalone local development environment.

The site is designed as a lightweight, high-performance marketing platform focused on:

- Commercial disinfecting services
- Residential fogging services
- Local SEO visibility
- Mobile-first user experience
- Fast static deployment
- Long-term maintainability

---
